# CursorFun.com

CursorFun.com 是一个专注于 Cursor IDE 相关资源的导航网站。网站采用现代化的设计风格，提供了丰富的 Cursor IDE 相关资源和教程。

## 特性

- 🎨 现代简约的设计风格
- 🌓 深色模式支持
- 📱 响应式布局，完美适配移动端
- 🌍 中英文双语支持（开发中）
- 💬 在线交流群

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- next-themes

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 部署

项目使用 Vercel 进行部署，每次推送到 main 分支会自动触发部署。
