# CursorFun.com 管理后台

这是 CursorFun.com 导航站点的管理后台，用于管理导航站点的内容，包括分类、导航项和群二维码。

## 功能概述

管理后台包括以下主要功能：

### 1. 仪表盘

- 显示导航站点的基本统计数据（分类数、导航项数等）
- 展示最近添加的导航项
- 提供快速操作入口

### 2. 分类管理

- 添加新分类
- 编辑现有分类（名称、描述、排序）
- 删除分类（会同时删除该分类下的所有导航项）
- 分类列表展示和排序

### 3. 导航项管理

- 添加新导航项
- 编辑现有导航项（标题、描述、链接、所属分类、排序）
- 删除导航项
- 按分类筛选导航项

### 4. 群二维码管理

- 查看当前群二维码
- 上传新的群二维码图片（支持预览）

## 数据存储

管理后台使用 zustand 进行状态管理，通过 localStorage 实现数据持久化。在实际生产环境中，您可能需要将其替换为真实的数据库存储。

目前的数据模型包括：

- 分类 (Category)
- 导航项 (NavItem)
- 二维码 (QRCode)

## 技术实现

- 前端框架：Next.js 14
- 状态管理：zustand + localStorage
- UI组件：自定义UI组件库
- 路由：App Router (Next.js 14)

1. 访问 `/admin` 路径进入管理后台
2. 使用左侧导航菜单切换不同的管理功能
3. 按照界面提示进行操作
