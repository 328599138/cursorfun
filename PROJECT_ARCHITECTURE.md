# CursorFun.com 项目架构

CursorFun.com 是一个专注于 Cursor IDE 相关资源的导航网站。本文档描述了项目的整体架构、代码结构和主要功能模块。

## 项目概览

- **前端框架**: Next.js 14 + TypeScript
- **样式方案**: Tailwind CSS + Apple Design 风格
- **状态管理**: zustand
- **主题支持**: next-themes (深色/浅色模式)
- **部署平台**: Vercel

## 设计风格

项目采用 Apple Design 设计风格：

- **字体**: 系统字体堆栈，优先使用苹果系统字体
- **颜色**: 采用苹果系统色板
  - 主色调：蓝色 (#007AFF)
  - 成功：绿色 (#34C759)
  - 警告：橙色 (#FF9500)
  - 错误：红色 (#FF3B30)
  - 信息：浅蓝 (#5AC8FA)
- **圆角**: 采用苹果标准圆角值
- **阴影**: 轻微的阴影效果，营造层次感
- **动画**: 平滑的过渡动画，使用苹果标准缓动函数
- **深色模式**: 完整支持系统深色模式

## 核心功能模块

### 1. 导航站点（前台）

前台是一个简洁的导航站点，展示分类整理好的 Cursor IDE 相关资源链接。

主要功能：
- 响应式布局，适配各种设备
- 深色/浅色模式切换
- 微信群二维码展示
- 分类导航系统

### 2. 管理后台

管理后台允许管理员管理导航站点的内容。

主要功能：
- **仪表盘**: 展示站点统计数据
- **分类管理**: 添加、编辑、删除分类
- **导航项管理**: 添加、编辑、删除导航项
- **二维码管理**: 更新微信群二维码

### 3. 数据管理

使用 zustand 进行状态管理，通过 localStorage 实现数据持久化。

数据模型：
- **分类 (Category)**: 用于组织导航项的分类
- **导航项 (NavItem)**: 具体的导航链接
- **二维码 (QRCode)**: 微信群二维码

## 前端组件

### 通用组件

- **NavCard**: 导航卡片组件
- **ThemeToggle**: 主题切换按钮
- **ChatGroup**: 微信群组件
- **Header**: 顶部导航栏
- **Footer**: 页脚组件

### 管理后台组件

- **布局组件**: 管理后台的整体布局
- **表单组件**: 数据编辑表单
- **列表组件**: 数据展示列表
- **操作组件**: 按钮、开关等交互组件
